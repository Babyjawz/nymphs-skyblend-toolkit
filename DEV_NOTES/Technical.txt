ðŸ§  Mode Detection

_detect_pbr(textures) determines whether to build a PBR or vanilla layout.

Detection is based on texture filename patterns (_rmaos.dds, _orm.dds, _metal.dds, etc.).

The user can override this with the Force Build (PBR) checkbox in the material panel.
